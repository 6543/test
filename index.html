Woodpecker CI

Woodpecker is a simple CI engine with great extensibility.

OpenSource and free

Woodpecker is and will be totally free for ever. As Woodpeckers source code is OpenSource you can contribute to help evolving the project.

Based on docker containers

Woodpecker uses docker containers to execute pipeline steps. If you need more than a normal docker image, you can create plugins to extend the pipeline features. How do plugins work?

Multi pipelines

Woodpecker allows you to easily create multiple pipelines for your project. They can even depend on each other. Check out the docs
